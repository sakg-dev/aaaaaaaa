An interactive knowledge graph for CS/dev concepts that helps people understand how things work “under the hood” instead of only learning how to use them. Users can search something high-level like React, Docker, HTTP, or databases and explore connected lower-level concepts (e.g., React → Virtual DOM → Browser DOM → JS engine → OS/CPU/binary), with explanations at each layer.
